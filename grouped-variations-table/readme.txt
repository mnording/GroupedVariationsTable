Allowing you to group variations in sleak tables on the product page

== Description ==
This plugin allows you to define a master attribute that your variable products uses.
It will then render a table that is grouping the variations based on that main attribute.

== Installation ==
Upload folder to your wp-content/plugins directory

Activate through admin panel
